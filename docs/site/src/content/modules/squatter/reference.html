Payload TLS Credential Slot

name: payload-tls-server
delivery: stamp-standard
purpose: tls-server
endpointRole: server
consumerType: payload
profile: tls-server
compatibilityTarget: portable-x509
projection: bundle
materialForm: private-bytes
remainderPolicy: zero-fill

The provider validates the complete hovel.pki.bundle/v1 and writes a versioned, integrity-bound SQPKI001 manifest into the generated PE. The payload loads the certificate chain and PKCS#8 key into statically linked wolfSSL. Mesh does not receive or terminate the TLS identity.
