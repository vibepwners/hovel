picblobs

picblobs is Hovel's position-independent payload blob provider module. It imports the original pic source tree under modules/picblobs, keeps the original tests and specifications, and exposes a Hovel JSON-RPC payload-provider shim.

Entry Points

Use User Guide Stage provider and payload artifacts, then link the module into a Hovel workspace. Build Build Notes Root Bazel configs, platform configs, staging, and provider binary output. Verify Testing Bazel tests, Python tests, payload tests, generated-file checks, and dist checks. Ref Reference Important files, module manifest, preserved original docs, and source specs.
