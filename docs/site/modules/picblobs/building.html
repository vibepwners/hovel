Building picblobs

picblobs uses the repository root .bazelrc. The imported modules/picblobs/.bazelrc is retained only as deprecated source context.

Bazel Targets

task picblobs:bazel-build
bazel build --config=linux_x86_64 //modules/picblobs/src/payload:hello
bazel build //modules/picblobs/provider:picblobs-provider

Staging

task picblobs:provider-stage
task picblobs:stage

The provider stage copies the Go payload-provider binary into the module package. The blob stage builds platform payloads and extracts release artifacts into the Python package tree.
